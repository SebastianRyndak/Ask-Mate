# Ask-Mate'


## About the app

Ask-Mate is an online forum where users can ask questions on various topics, give answers and mark the ones they think are best. 


## Technologies

Backend technologies:

- Python 3.10
- Flask 2.0.2
- PostgreSQL 14

Frontend technologies:

- Vanila JS 
- HTML 5
- Bootstrap 5.1.3
- CSS
